# zhm-leyu-resource

本仓库用于归档和发布多个独立的 HTML 页面，旨在提供一种结构化的方式管理和共享前端资源。

## 项目简介

这是一个通用型 HTML 页面资源仓库，不针对任何特定域名或具体网站。仓库中的页面均为独立、完整的 HTML 文件，可直接在浏览器中打开使用。适用于存放各类静态页面、工具页面、信息展示页等。

## 目录说明

```
zhm-leyu-resource/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面依赖的静态资源（CSS、JS、图片等）
├── docs/           # 文档或辅助说明文件（可选）
└── README.md       # 本文件
```

- `pages/`：每个子目录对应一个独立页面，包含其主 HTML 文件及相关资源。
- `assets/`：可供多个页面共享的公共资源文件。
- `docs/`：可存放页面使用说明、更新日志等文档。

## 页面归档说明

仓库中的每个 HTML 页面均独立归档，遵循以下原则：

- 每个页面应有清晰的命名，并放置在 `pages/` 下的独立子目录中。
- 页面内引用的资源（如样式、脚本）尽量使用相对路径，确保移动或复制时仍可正常访问。
- 页面内容不包含任何营销或诱导点击信息，仅以提供功能或信息展示为目的。

如需添加新页面，请在 `pages/` 下新建目录，并放置主 HTML 文件及所需资源。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下对应目录中的 HTML 文件。
3. 也可部署至任意静态服务器进行访问。
